# React Quiz App
A modern React application that provides an interactive quiz experience with timers, answer tracking, and a summary of results. The app delivers a smooth user experience with clear feedback for correct, incorrect, and skipped answers.

## Features
- Multiple-choice quiz with shuffled answers
- Timer for each question
- Track answered, skipped, correct, and wrong questions
- Summary page with performance statistics
- Interactive and responsive UI

## Tech Stack
- React (Vite)
- useState, useCallback, useEffect hooks
- Context API (for optional state management)
- CSS Modules or plain CSS

## Getting Started
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## How It Works
- Questions are displayed one at a time with a countdown timer.

- Users can select an answer or skip a question.

- The app tracks correct, wrong, and skipped answers in real-time.

- After the quiz, a summary page displays performance statistics and a detailed answer review.
